08
📏 Smart Distance Alert System
Arduino UNO HC-SR04 16×2 LCD Buzzer Breadboard

An Arduino-powered smart distance alert system that uses an HC-SR04 ultrasonic sensor to measure real-time distance and displays it live on a 16×2 LCD screen. When an object enters the danger zone, the buzzer triggers an instant alert. The LCD shows both the distance in cm and a status label — NEAR when something is close, FAR when the path is clear. Perfect for obstacle detection, parking assistance, and safety automation.

🦕 Dino Runner — Arduino OLED Game
Arduino UNO OLED Display Push Button Breadboard Game Dev

A fully playable Dino Runner game built entirely on Arduino — no computer screen needed! The pixel-art dinosaur runs automatically on a crisp 128×64 OLED display and you press a single physical button to jump over incoming obstacles.
